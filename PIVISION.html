MOD-01 Pastikan Fondasi AF Sudah Siap PREREQ
// TUJUAN: Sebelum buka PI Vision, pastiin data di AF udah ada dan benar
  • Buka PI System Explorer (PSE) dari Start Menu
  • Navigasi ke AF Database yang kamu pakai (contoh: PlantSimulation)
  • Expand Elements → pilih asset target (contoh: Pump_101)
  • Lihat tab Attributes → cek attribute yang mau ditampilkan, pastikan ada field PI Point Data Reference
  • Klik kanan attribute → pilih "Preview" → konfirmasi data muncul dengan nilai valid
✅ CHECKPOINT: Kalau preview kosong atau error, masalahnya ada di AF/tag mapping — BUKAN di PI Vision. Selesaikan dulu sebelum lanjut.
MOD-02 Buka PI Vision & Buat Display Baru CORE
// TUJUAN: Setup kanvas kerja di PI Vision
  • Buka browser → akses PI Vision: http://<servername>/PIVision
  • Klik tombol "New Display" (icon + atau dari menu utama)
  • Masuk ke Design Mode — klik icon pensil/edit di toolbar atas
  • Kasih nama display yang deskriptif → klik Save sebelum mulai nambah simbol
⚠️ PENTING: Design Mode HARUS aktif untuk bisa drag simbol ke display. Kalau drag nggak bisa, cek dulu apakah sudah dalam Design Mode.
MOD-03 Cari Data AF di Assets Pane CORE
// TUJUAN: Connect PI Vision ke AF Database dan temukan attribute target
  • Di panel kiri, klik tab "Assets" — bukan tab "Tags"
  • Klik dropdown pilih AF Server → pilih server yang benar (contoh: PISYSTEM)
  • Pilih AF Database yang berisi asset kamu (contoh: PlantSimulation)
  • Browse tree: Database → Element → Asset sampai kelihatan daftar Attributes
  • Alternatif: gunakan Search bar di atas Assets pane → ketik nama attribute atau element
💡 TIP: Kalau mau cari by PI Tag langsung tanpa struktur AF, pindah ke tab "Tags" dan search by tag name. Tapi untuk best practice, selalu gunakan AF attribute.
MOD-04 Buat Trend Symbol CORE
// TUJUAN: Drag attribute AF ke display sebagai grafik time-series
  • Di Symbol Gallery (atas Assets pane), pastikan icon Trend terpilih — icon grafik garis, default pertama
  • Dari hasil search/browse di Assets pane, klik + drag attribute ke area display kosong
  • PI Vision otomatis insert Trend Symbol dengan trace pertama dari attribute itu
  • Untuk tambah trace ke-2, ke-3: drag attribute lain langsung ke dalam trend yang sudah ada
  • Resize trend dengan drag di sudut/tepi kotak — posisikan sesuai kebutuhan layout
✅ HASIL: Satu Trend Symbol bisa isi beberapa attribute sekaligus (multi-trace). Ideal untuk compare beberapa parameter seperti temp, pressure, dan flow dalam satu grafik.
MOD-05 Konfigurasi Trend CORE
// TUJUAN: Rapikan tampilan trend biar informatif dan mudah dibaca
  • Klik kanan pada trend → pilih "Configure Trend" untuk buka panel konfigurasi
  • Set konfigurasi berikut sesuai kebutuhan:
Opsi Rekomendasi Keterangan
Plot Title Isi nama jelas Contoh: "Lube Oil Temperature"
Time Range Display time range Sinkron dengan Time Bar display
Scale Type Multiple / Single Multiple jika unit beda, Single jika sama
Scale Range Autorange of dynamic values Otomatis fit ke nilai aktual
Trace Color Bedain tiap trace Hindari warna serupa dalam satu trend
Legend Label Nama manusiawi Ganti raw tag name jadi readable
  • Klik tanda panah atas-kanan (X) panel Configure Trend untuk menutup
  • Klik Save display
MOD-06 Set Time Range Display CORE
// TUJUAN: Kontrol window waktu yang ditampilkan di semua simbol
  • Keluar dari Design Mode terlebih dahulu — klik icon Monitor Operations
  • Lihat Time Bar di bagian bawah display
  • Klik dropdown waktu → pilih range preset: Last 8 Hours, Last 1 Day, dll
  • Untuk custom: klik langsung pada start time dan end time → edit manual
  • Klik tombol "Now" untuk kembali ke real-time mode dengan data live
💡 NOTE: Mengubah Time Range di Time Bar akan update SEMUA trend yang dikonfigurasi dengan "Display time range". Trend dengan "Custom time range" tidak ikut berubah.
MOD-07 Monitor & Analisis Trend (View Mode) CORE
// TUJUAN: Gunakan fitur interaktif bawaan PI Vision untuk analisis data
  • Klik "Monitor Operations" (icon mata) → keluar dari Design Mode ke View Mode
  • Trend Cursor: klik area dalam trend → muncul garis vertikal dengan nilai + timestamp di tiap trace. Cursor otomatis sinkron ke semua trend di display
  • Pan timeline: arahkan cursor ke bagian bawah trend → drag kiri/kanan untuk geser waktu maju/mundur
  • Zoom in: klik + drag area dalam trend → trend zoom ke rentang waktu yang dipilih. Semua simbol di display ikut update
  • Undo zoom: tekan CTRL+Z untuk kembali ke zoom sebelumnya
  • Klik "Now" di Time Bar untuk kembali ke tampilan real-time
MOD-08 Ad Hoc Trend (Analisis Cepat) BONUS
// TUJUAN: Buat trend cepat untuk investigasi tanpa perlu bikin display dulu
  • Dari halaman utama PI Vision, klik "New" → pilih "Ad Hoc Trend"
  • Search dan drag attribute langsung ke canvas ad hoc trend
  • Analisis data sesuai kebutuhan — semua fitur trend tersedia
  • Kalau mau simpan hasilnya: klik "Convert to Display" → simpan sebagai display permanen
  • Untuk share ke kolega: klik "Share" → copy URL yang dihasilkan
💡 USE CASE: Ad Hoc Trend ideal buat investigasi cepat pas ada alarm atau anomali — langsung buka, drag beberapa tag, bandingkan, lalu close. Tidak perlu save jika hanya untuk analisis sementara.
✓ Checklist Praktik
01 AF attribute sudah linked ke PI Tag dan ada datanya (verifikasi via Preview di PSE)
02 Bisa akses PI Vision dari browser tanpa error
03 Berhasil buat display baru dan masuk ke Design Mode
04 Browse AF di Assets pane dan menemukan attribute target
05 Drag attribute → Trend Symbol muncul di display dengan trace data
06 Tambah attribute ke-2 ke trend yang sama (multi-trace berhasil)
07 Configure Trend: kasih judul, bedakan warna trace, set time range
08 Exit Design Mode → trend cursor berfungsi dan bisa baca nilai di titik waktu tertentu
09 Berhasil zoom in dan pan di trend timeline
10 Display tersimpan dan bisa dibuka kembali
⚡ Common Errors & Fix
ERR-01 Trend kosong / flatline — PI Tag belum menerima data dari interface/OPC. Cek status OPCInt.exe atau Connector, dan verifikasi tag ada valuenya di PI SMT.
ERR-02 AF attribute preview error — Data Reference tidak benar. Masuk PSE → attribute → pastikan PI Tag name valid dan server-nya benar.
ERR-03 Tidak bisa drag simbol ke display — Belum masuk Design Mode. Klik icon pensil/edit di toolbar PI Vision terlebih dahulu.
ERR-04 AF Database tidak muncul di Assets pane — Pilih AF Server yang benar dulu dari dropdown. Kalau server tidak muncul, cek koneksi PI AF ke PI Vision.
ERR-05 Data muncul tapi nilai "No Data" / "Shutdown" — Kemungkinan time range display terlalu jauh ke masa lalu atau future. Set time range ke "Last 8 Hours" dan klik "Now".
